[Erle-brain2] console 명령어 모음

유저 네임 변경 (꼬일 수 있으니 하지 말것)
http://askubuntu.com/questions/34074/how-do-i-change-my-username
sudo su -
usermod -l newUsername oldUsername
usermod -d /home/newhomeDir -m newUsername # 홈폴더 변경

유저 패스워드 변경
sudo passwd erle

데비안 GUI 경우
startx --> 안 됨. 최초 한번만 실행되고 그 후로는 GUI disabled 상태.
reset-erle --> 다시 GUI 보고 싶을 경우

호스트 네임 변경
http://www.howtogeek.com/167195/how-to-change-your-raspberry-pi-or-other-linux-devices-hostname/
sudo vim /etc/hostname # 호스트 이름 수정 후 저장
sudo vim /etc/hosts # 127.0.1.1 옆의 이름 호스트이름과 동일하게 수정
sudo /etc/init.d/hostname.sh
sudo reboot

erle-brain에서 다른 wifi 네트워크로 접속(client mode)

wpa_passphrase bigdata bigdata1 > erle.conf
sudo mv erle.conf /etc/wpa_supplicant/
vim /etc/network/interfaces 아래 문구 추가
auto wlan0
allow-hotplug wlan0
iface wlan0 inet manual
wpa-conf /etc/wpa_supplicant/erle.conf
