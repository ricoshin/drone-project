1. 포켓 스핑크스pocketsphinx 라이브러리

포켓 스핑크스 설정 및 파이썬 바인딩 설치
sudo apt-get install pocketsphinx-hmm-wsj1리
sudo apt-get install pocketsphinx-lm-wsj
sudo apt-get install python-pocketsphinx


오프라인 음성 인식 실행
wav 파일 녹음 & 실행

파이썬 코드

실행 명령>
python 경로/pythonPocketSphinxTest.py 경로/웨이브파일이름.wav


실시간 음성 인식 실행
파이썬 코드

실행 명령>
python 경로/pocketsphinx_gst.py



2. 줄리우스julius 라이브러리

줄리우스 음성 인식기와 파이썬 모듈 설치
sudo apt-get install julius
sudo apt-get install pulseaudio-utils
sudo apt-get install osspd-alsa
sudo alsa force-reload
sudo alsa force-reload
wget https://bootstrap.pypa.io/ez_setup.py
압축파일 풀고 해당폴더에서,
sudo python ez_setup.py
sudo python setup.py install
wget http://www.repository.voxforge1.org/downloads/software/julius-.3.5.2-quickstart-linux.tgz
압축파일 풀기

음성인식 실행
실행 명령>(압축파일 푼 폴더에서)
padsp julius -input mic -C julian.jconf

파이썬과 연결해 음성인식 실행
실행 명령>(압축파일 푼 폴더에서)
padsp julius -module -input mic -C julian.jconf

파이썬 줄리우스 클라이언트 코드

python 경로/pyjulius_test.py
(but "ImportError: No module named pyjulius " 에러 뜸;)
