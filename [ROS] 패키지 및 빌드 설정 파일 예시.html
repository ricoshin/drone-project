# catkin_python_setup()

##############################
## ROS 메세지 / 서비스 / 액션 선언 ##
##############################

## 이 패키지 내부에 있는 메세지 / 서비스 / 액션을 선언하고 빌드하려면 다음 단계를 따를 것.
## (당신의 패키지가 하나 이상의 메세지 의존성을 갖는 다면 다음을 수행해야 함)

## 지금부터 당신의 패키지에서 사용된 메세지 타입(메세지/서비스/액션) 세트를 
## "MSG_DEP_SET" (message dependency set) 이라고 표시하겠다.
## 실제로는 std_msgs, common_msgs(?), sensor_msgs, actionlib_msgs 등을 포함하는 세트일 것이다.

## package.xml 파일에서 다음을 추가한다.
## * <build_depend> 태그에 "message_generation"을 추가
## * <run_depned> 태그에 "message_runtime"을 추가
## * <build_depend>와 <run_depend> 태그에 MSG_DEP_SET의 모든 패키지를 추가
##    ** 사실상 "MSG_DEP_SET"을 추가하면 message_runtime이 포함되는 것과 같지만 확실함을 위해 추가한다.

## CMakeLists.txt 파일에서 다음을 추가한다.
## * find_package(catkin REQUIRED COMPONENTS ...)에 "MSG_DEP_SET"의 모든 패키지와 "message_generation"을 추가
## * catkin_package(CATKIN_DEPENDS ...)에 "MSG_DEP_SET"의 모든 패키지와 "message_runtime"을 추가
## * add_*_files() 섹션에서 모든 .msg/.srv/.action 파일 리스트를 추가
## * generate_messages(DEPENDENCIES ...)에 "MSG_DEP_SET"의 모든 패키지를 추가

## 'msg' 폴더에 있는 메세지파일(*.msg) 리스트
add_message_files(
   FILES
   msgTutorial.msg
)

## 'srv' 폴더에 있는 서비스파일(*.srv) 리스트
add_service_files(
   FILES
   srvTutorial.srv
)

## 'action' 폴더에 있는 액션파일(*.action) 리스트
#add_action_files(
#   FILES
#   Action1.action
#   Action2.action
# )

## 위에서 추가한 메시지들을 생성하기 위해 필요한 모든 depedency들 리스트
## 즉, "MSG_DEP_SET"의 모든 패키지 리스트
generate_messages(
   DEPENDENCIES
   std_msgs
)

##################
## catkin 특수 설정 ##
##################
## catkin_package() 명령어(함수 or 매크로)는 당신의 패키지를 위한 cmake config files을 만든다.
## 추후에 다른 프로젝트에서 이 패키지를 의존할 때(불러들일 때) 넘겨주어야 할 것들을 선언하라.
## 참고) A(현재 패키지)에서 export한 것을 B(다른 ROS 패키지)에서 import할때 B는 A를 의존한다.

## INCLUDE_DIRS: 당신의 패키지가 헤더파일을 포함하고 있을 경우 그 경로(include/폴더)를 명시한다.
## LIBRARIES: 다른 프로젝트에서 의존할 수 있도록 이 프로젝트에서 만든 라이브러리
## CATKIN_DEPENDS: 현재 프로젝트를 의존하는 다른 프로젝트에서도 동일하게 필요로 하는 catkin_package
## DEPENDS: 현재 프로젝트를 의존하는 다른 프로젝트에서도 동일하게 필요로 하는 system dependencies
catkin_package(
  INCLUDE_DIRS include
  LIBRARIES oroca_ros_tutorials
  CATKIN_DEPENDS roscpp std_msgs
  DEPENDS system_lib
)
