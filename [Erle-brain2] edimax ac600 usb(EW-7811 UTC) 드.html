[Erle-brain2] edimax ac600 usb(EW-7811 UTC) 드라이버 설치 방법(리눅스)

edimax ac600 usb : 드론과 같이 배송된 wifi adapter

복사/붙여넣기
------------------------------------------------------------------------------------------
sudo apt-get update &&
sudo apt-get install linux-headers-generic build-essential git &&
git clone https://github.com/gnab/rtl8812au.git &&
cd ~/rtl8812au &&
make &&
sudo make install &&
sudo modprobe 8812au
------------------------------------------------------------------------------------------

참고 :
  • 앞의 명령어가 성공해야 다음에 이어지는 명령어를 수행 =>  &&
  •  앞의 명령어가 실패하면 다음에 이어지는 명령어를 수행 =>  ||
  •  명령어의 성공유무와 상관없이 명령어 리스트를 순차적으로 실행 =>  ;
# git clone : 원격저장소 복제
     git clone <repository(원격저장소 url)> <directory(복제물을 넣을 폴더명. 없으면 원 폴더명 그대로, 있으면 새로운 폴더명 아래 내용물 저장됨)>
     ex) git clone https://github.com/gnab/rtl8812au.git wifi
     => 현재 폴더의 wifi라는 폴더 아래 해당 url의 파일들이 복제 저장됨

# make : 소스 컴파일(소스파일을 실행가능한 파일로 만들어줌).
복제해온 파일 중 makefile(프로그램의 빌드 과정을 표준 문법으로 기술한 파일. 컴파일 순서나 각 소스파일 간 의존성 정보 기록됨) 있음. 이를 참고해 make하게 됨.

# make install : 컴파일된 파일을 시스템에됨 설치(적절한 위치로 파일을 이동시키고 bin 폴더에 링크파일 생성).

# modprobe : 모듈 적재(단일모듈, 의존성 있는 여러 모듈, 특정 디렉토리 모든 모듈 적재 가능). 옵션 사용시(-r) 모듈 제거도 가능. '기호=값' 형식을 이용해 매개변수도 전달 가능. (그래서 insmod나 rmmod보다 권장됨)
      사용법: modeprobe [옵션] 모듈명 [기호=값]
