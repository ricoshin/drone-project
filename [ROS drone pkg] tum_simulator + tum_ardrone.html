[ROS drone pkg] tum_simulator + tum_ardrone

패키지 내용

1.cvg_sim_gazebo : 오브젝트 모델, 센서 모델, 쿼드로콥터 모델, 비행 환경 정보, 각각의 오브젝트와 환경별 launch파일들을 포함.
2.cvg_sim_gazebo_plugins : 쿼드콥터 모딩을 위한 gazebo플러그인.
                                               quadrotor_simple_controller는 로봇 모션에 사용되고 /ardrone/navdata같은 네비게이션 정보를 내보낸다.
                                               다른 플러그인들은 IMU 센서, 초음파 센서, GPS 센서와 같은 쿼드로콥터에 있는 센서들을 위한 것들이다.
3.message_to_tf : ros node를 생성하기 위한 패키지. /ground_truth 토픽을 /tf토픽으로 옮겨준다.
4.cvg_sim_msgs : 시뮬레이터를 위한 메세지 형식을 포함한다.




시뮬레이션 상의 데이터(실제 AR.Drone에서 보내는 데이터와 같은 이름으로 구현)
  • Front camera information and image data
  • Bottom camera information and image data
  • Activated camera information and image data
  • IMU data
  • tf data
  • Navigation data

navdata_구현 정보
  • message time stamp
  • message frame id
  • fly mode (partly)
  • battery percentage
  • position, rotation
  • velocity
  • acceleration

navdata_미구현 정보
  • magnet info
  • pressure
  • wind information
  • tags

시뮬레이션에서 구현되는 센서들
  • camera sensors
  • altitude sensor
  • IMU sensor


설치 방법

cd ~/catkin_ws     # 자기 자신의 캣킨 workspace로 들어간다.
mkdir sim_src      # 기존의 src폴더와 충돌 방지 및 빌더 에러를 피하기 위해 따로 만들어 줌.
cd sim_src         # 문제가 없다면 기존 src 폴더에서 빌드해도 관계없음.

git clone https://github.com/AutonomyLab/ardrone_autonomy.git             # AR-drone 드라이버
git clone https://github.com/occomco/tum_simulator.git                    # 시뮬레이터 
git clone https://github.com/tum-vision/tum_ardrone.git -b indigo-devel    # tum_ardrone 설치

cd ..
rosdep install --from-paths sim_src --ignore-src --rosdistro indigo -y     # 의존 패키지 설치
catkin_make --source sim_src      # catkin 빌드
source devel/setup.bash           # 환경 재설정
실행 방법 (각각 다른 터미널에서 실행)

roscore                                                # 마스터 실행 (Master URI와 호스트 설정은 알아서...)
roslaunch cvg_sim_gazebo ardrone_testworld.launch      # 시뮬레이터 실행
roslaunch tum_ardrone tum_ardrone.launch               # tum_ardrone 실행

터미널 커맨드로 조작해보기

  • take off


rostopic pub -1 /ardrone/takeoff std_msgs/Empty
  • land


rostopic pub -1 /ardrone/land std_msgs/Empty
  • switch camera


rosservice call /ardrone/togglecam
  • motion (반대방향은 부호만 x/y/z 부호만 반대로)
# fly forward
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 1.0, y: 0.0, z: 0.0}, angular: {x: 0.0,y: 0.0,z: 0.0}}'
# fly to left 
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 0.0, y: 1.0, z: 0.0}, angular: {x: 0.0,y: 0.0,z: 0.0}}'
# fly to right 
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 0.0, y: -1.0, z: 0.0}, angular: {x: 0.0,y: 0.0,z: 0.0}}'
# fly up 
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 0.0, y: 0.0, z: 1.0}, angular: {x: 0.0,y: 0.0,z: 0.0}}'
# counterclockwise rotation
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 0.0, y: 0.0, z: 0.0}, angular: {x: 0.0,y: 0.0,z: 1.0}}'
# stop
rostopic pub -r 10 /cmd_vel geometry_msgs/Twist  '{linear:  {x: 0.0, y: 0.0, z: 0.0}, angular: {x: 0.0,y: 0.0,z: 0.0}}'

센서 값 읽어오기

rosrun image_view image_view image:=/ardrone/image_raw
# The front camera
rosrun image_view image_view image:=/ardrone/front/image_raw
# The buttom camera
rosrun image_view image_view image:=/ardrone/bottom/image_raw
# The height sensor
rostopic echo /sonar_height
#The navigation info
rostopic echo /ardrone/navata

토픽 확인하기

rostopic echo /cmd_vel
rostopic echo /ardrone/land
rostopic echo /ardrone/reset
rostopic echo /ardrone/takeoff

키보드 조작키
D : Land
S : Takeoff
I : forward
K : backward
U : counterclockwise rotation (turn left)
O : clockwise rotation (turn right)
J : left
L : right
Q : up
A : down
