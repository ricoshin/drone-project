데이터 있는 마지막 부분 확인하기


$ sudo fdisk -l /dev/mmcblk0


 


전송 받을 곳에서 실행


$ nc -l [포트번호] | dd of=[파일명] bs=512


ex) nc -l 18000 | dd of=jack.img bs=512

포트번호 18000번을 열고 기다린다 | block size(단위 : byte) 512, 파일명 jack.img 로 복사 받는다.




전송할 곳에서 실행


$ sudo dd if=[전송할_드라이브] bs=512 count=[데이터_마지막_부분] | nc [전송할_IP] [포트번호]


ex) $ sudo dd if=/dev/mmcblk0 bs=512 count=14999551 | nc 192.168.0.13 18000

[/dev/mmcblk0] (여기선 sd카드)를 [block size=512]로 처음부터 [14999551]까지

[ip 192.168.0.13] [port 18000] 으로 복사한다.

라즈베리파이에 USB 메모리 연결하여 백업할 경우

dd if=[백업할_드라이브] of=[복사할 경로] bs=512 count=[데이터_마지막_부분]

ex) dd if=/dev/mmmcblk0 of=/media/pi/usb/pib.img bs=512 count=123456789

$ df -h 명령으로 usb 경로 확인
/media/pi/usb : usb 메모리

sd카드 어댑터로 연결(usb)해 복사할 때:

dev/ 아래 목록에서 usb 장치 이름 확인(예: sdc)
     (gparted에서 sd카드 연결장치 이름 확인가능) 
sudo fdisk -l /dev/sdc (장치명이 sdc일 때) -> End의 숫자 확인(예: 16566271)
sudo dd of=/home/[사용자명]/jack.img if=/dev/sdc bs=512 count=16566271 (End에서의 숫자 입력)
